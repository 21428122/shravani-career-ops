## How this works

- **Scanner** (`../scan.mjs` in the Career-Ops install) polls Greenhouse / Ashby / Lever APIs for new PM-intern-friendly roles every day.
- **Classifier** matches each posting to a niche from `templates/niches.json` based on company + title keywords.
- **Generator** fills `templates/resume-template.html` with niche-specific headline + skills and compiles a PDF via Playwright.
- **This repo** holds every tailored PDF so you can grab the right one for each application.
