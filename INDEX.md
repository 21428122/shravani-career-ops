## How this works

- **Scanner** polls Greenhouse / Ashby / Lever APIs daily for new PM/APM/Intern roles.
- **Classifier** matches each posting to a niche using keyword matching on company + title.
- **Generator** fills the HTML template with niche-specific headline + skills and renders a 1-page PDF via Playwright.
- **Contacts** for each company are in [`data/contacts.json`](data/contacts.json); the top contact appears in the table above.
- **Outreach templates** are in [`outreach/`](outreach/) — pick the one matching the niche.
