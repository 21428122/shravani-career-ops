# Apply Playbook — Step-by-Step

You wake up, you open this repo, you want to apply to a company. Here's exactly what to do.

## For each application (takes ~12-15 minutes per company)

### Step 1 — Pick the company from [INDEX.md](INDEX.md)

The table has everything: company, role, niche, your tailored PDF, apply link, the one person to DM first, and which DM template to use.

### Step 2 — Apply through the portal

1. Click the **Apply** link in INDEX.md.
2. Download the matching PDF from the `pdfs/` folder (or right-click → "Save Link As" on the Resume column).
3. Fill out the company's application form.
4. **Upload the tailored PDF** (not the Base one) — the headline names the company, which reviewers notice in the first 5 seconds.
5. Submit.

### Step 3 — DM the "DM first" contact

Every row has a named LinkedIn profile. Open it, and:

1. Click **Connect**.
2. Click **Add a note** (important — do not send a blank request).
3. Paste the connection note from the matching template (under 200 characters).
4. Customize the first 5-10 words so it reads personally — name them, reference something specific about the company's recent news / product launch / the person's own LinkedIn posts.
5. Send.

### Step 4 — After they accept, send the follow-up DM

1. DM them with the full follow-up message from the template.
2. Attach the tailored PDF.
3. End with a single, low-friction ask: "Would a 15-minute call be useful?" or "Is the hiring manager open to referrals?"

### Step 5 — Log it

Open [tracker.md](tracker.md) and add a row. If that file doesn't exist yet, the format is:

```
| Date | Company | Applied? | DM sent? | Reply? | Interview? | Notes |

```

## Rules of thumb

- **One custom DM per company per week.** If no reply after 5 business days, follow up once. If still no reply, move on.
- **Never send the same DM to two people at the same company in the same week** — they talk.
- **Always attach the company-specific PDF** — not the Base one. Headline matters.
- **Track every send.** 15 applications without tracking = chaos. 15 applications with tracking = a clean pipeline.
